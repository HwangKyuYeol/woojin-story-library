# WooJin Story Library

우진이 성장 도서관 - 기차 여행 + 제주도 여행 Part 1 만화 동화책 통합 버전입니다.

## GitHub 업로드 방법

1. ZIP 압축을 풉니다.
2. `woojin-story-library` 저장소 루트에 아래 파일/폴더를 그대로 업로드합니다.
   - `index.html`
   - `css/`
   - `js/`
   - `assets/`
   - `README.md`
3. GitHub Pages 설정에서 `main` / `/root` 로 배포합니다.

## 기능

- 도서관 메인 화면
- 기차 여행 책 선택
- 제주도 여행 Part 1 책 선택
- 책 읽는 중 `도서관으로` 버튼
- 좌우 클릭/스와이프 페이지 넘김
- 이미지 엑박 방지를 위한 상대 경로 정리
- 아이패드/모바일 화면 최적화
- 소리 켜기 버튼 후 페이지 효과음
